## How To Read This GitHub

Public repos are inspection surfaces, not a dump of every working object. Some systems remain private while they are still provenance, operations, or internal research infrastructure.

The important boundary is:

- Papers and public READMEs are citeable research surfaces.
- Instruments are runnable or inspectable software surfaces.
- Private vaults, sensor systems, and unpublished drafts are not implied by public links unless a repo says so directly.
- A date on a README means the public framing was reviewed on that date, not that every internal artifact changed that day.
