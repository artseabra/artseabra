# Art Seabra

Reviewed: 2026-07-08

## Boundary

Public here means inspectable, readable, runnable, or deliberately routed. It does not mean the private vault, unpublished drafts, internal sensor systems, or provenance repos are public dependencies.

The main public research line is **The Ærr Trinity**. The main software line is **ÆrrLedger**. The rest of the page exists to show how those ideas become instruments and shipped surfaces.
